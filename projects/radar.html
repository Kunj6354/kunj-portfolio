Radar Detection & Terrain-Aware Simulation

A Python and Angular platform for CAT048 and CAT062 radar-data visualization, replay, live UDP processing, diagnostics, and physics-based terrain-aware radar performance simulation.

Project overview

This project combines two related engineering areas: processing and visualizing radar-data files, and independently implementing a radar performance simulator in Python.

The simulator is designed around geometry, Earth curvature, propagation horizon, terrain masking, target characteristics, signal-to-noise behaviour, probability of detection, and deterministic validation—not fixed-radius circles.

Engineering principles

Terrain-aware coverage

Detection coverage is derived from geometry and environmental constraints so elevated and masked terrain can produce irregular visible regions.

Deterministic testing

Stochastic behaviour uses controlled seeds so expected detection and measurement behaviour can be tested repeatedly.

Independent implementation

MATLAB and radar-toolbox documentation may be used as conceptual reference material, but the project does not depend on MATLAB runtime or proprietary source code.

Incremental validation

New physics and runtime features are added in small phases and validated before expanding the model.

Outcome

The project now provides a structured foundation for radar-data processing and progressively higher-fidelity simulation. Its architecture keeps physics, runtime, APIs, testing, and visualization separate enough to evolve safely.

This case study presents general software, physics, testing, and visualization concepts. It does not expose protected operational parameters or implement autonomous engagement logic.
